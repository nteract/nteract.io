# nteract.io

Landing page for [nteract desktop](https://github.com/nteract/desktop) — native interactive notebooks.

## Development

```bash
git clone https://github.com/nteract/nteract.io
cd nteract.io
pnpm install
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Stack

- [Next.js 15](https://nextjs.org/) with App Router
- [Tailwind CSS](https://tailwindcss.com/)
- [pnpm](https://pnpm.io/)

## Deployment

The site is deployed on [Vercel](https://vercel.com/). Merging to `main` triggers automatic deployment to [nteract.io](https://nteract.io).
